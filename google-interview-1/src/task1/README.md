In the data file you can find the representation of a linear route with different boroughs on it after each other, in a city. Between each borough there is one step distance. The boroughs come after each other in a linear way. In each borough there are different facilities, which are represented with true value in the data file if the facility can be found in that particulat borough. For every borough, every type of facility is present either with true or false value.

Question

Given different interests of facilities' presence, find the best borough(s) for a set of interests where one person has to travel the shortest to be able to reach each of the facilities they are interested in.
